В мобильной версии вы можете опробывать HTML теги в нашей песочнице)

Заголовки

Заголовки определяются с помощью тегов <h1> до <h6>. Пример:

<h1>Это заголовок 1 уровня</h1>

Параграфы

Параграфы текста определяются с помощью тега <p>. Пример:

<p>Это пример параграфа текста.</p>

Это пример параграфа текста.

Блоки (div)

Блочные элементы создаются с помощью тега <div>. Пример:

<div>Это блочный элемент.</div>
Это блочный элемент.

Навигация

Навигационные ссылки определяются с помощью тега <nav>. Пример:

<nav><ul><li><a href="#">Ссылка</a></li></ul></nav>

Списки

Списки создаются с помощью тегов <ul> и <li>. Пример:

<ul><li>Элемент списка</li></ul>
  • Элемент списка

Классы и ID

Классы и ID используются для стилизации и идентификации элементов. Пример:

<div class="example">Класс</div><div id="unique">ID</div>
Класс
ID

Ссылки (href)

Ссылки создаются с помощью тега <a> и атрибута href. Пример:

<a href="https://example.com">Это ссылка</a>
Это ссылка

Изображения (src)

Изображения вставляются с помощью тега <img> и атрибута src. Пример:

<img src="image.jpg" alt="Пример изображения">

Формы (input)

Формы создаются с помощью тега <form> и различных элементов <input>. Пример:

<form><input type="text" placeholder="Введите текст"></form>

Горизонтальные линии (hr)

Горизонтальные линии создаются с помощью тега <hr>. Пример:

<hr>

Кнопки (button)

Кнопки создаются с помощью тега <button>. Пример:

<button>Нажми меня</button>

Таблицы (table)

Таблицы создаются с помощью тега <table> и его подэлементов <tr>, <th> и <td>. Пример:

<table>\n  <tr>\n    <th>Заголовок</th>\n    <th>Заголовок</th>\n  </tr>\n  <tr>\n    <td>Ячейка</td>\n    <td>Ячейка</td>\n  </tr>\n</table>

Цитаты (blockquote)

Цитаты создаются с помощью тега <blockquote>. Пример:

<blockquote>Это пример цитаты.</blockquote>
Это пример цитаты.

Код (code)

Для вставки кода используется тег <code>. Пример:

<code>console.log('Hello, world!');</code>
console.log('Hello, world!');

Полужирный текст (strong)

Полужирный текст создаётся с помощью тега <strong>. Пример:

<strong>Это полужирный текст.</strong>
Это полужирный текст.

Видео (video)

Видео вставляются с помощью тега <video> и атрибутов src, controls, autoplay, loop. Пример:

<video src="video.mp4" controls autoplay loop></video>

Аудио (audio)

Аудио вставляются с помощью тега <audio> и атрибутов src, controls, autoplay, loop. Пример:

<audio src="audio.mp3" controls autoplay loop></audio>
