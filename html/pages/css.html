Испытание

В мобильной версии вы можете опробывать CSS свойства в нашей песочнице)

color

Задает цвет текста.

color: red; /* Устанавливает цвет текста красным */
color: #00FF00; /* Устанавливает цвет текста с помощью шестнадцатеричного значения */
color: rgb(0, 0, 255); /* Устанавливает цвет текста с помощью значения RGB */

font-size

Задает размер шрифта.

font-size: 16px; /* Устанавливает размер шрифта в 16 пикселей */
font-size: 1.5em; /* Устанавливает размер шрифта в 1.5 раза больше родительского элемента */
font-size: 120%; /* Устанавливает размер шрифта в 120% от родительского элемента */

font-weight

Задает толщину (жирность) шрифта.

font-weight: normal; /* Обычный вес шрифта */
font-weight: bold; /* Жирный вес шрифта */
font-weight: 700; /* Числовое значение, представляющее жирный вес шрифта */

line-height

Задает высоту строк текста.

line-height: 1.5; /* Устанавливает высоту строк в 1.5 раза больше размера шрифта */
line-height: 20px; /* Устанавливает высоту строк в 20 пикселей */

text-decoration

Определяет оформление текста.

text-decoration: none; /* Без оформления текста */
text-decoration: underline; /* Подчеркнутый текст */
text-decoration: line-through; /* Перечеркнутый текст */
text-decoration: overline; /* Надчеркнутый текст */

text-align

Определяет горизонтальное выравнивание текста.

text-align: left; /* Выравнивание текста по левому краю */
text-align: center; /* Центрирование текста */
text-align: right; /* Выравнивание текста по правому краю */
text-align: justify; /* Растягивает строки так,
 чтобы каждая строка имела одинаковую ширину */

font-style

Определяет стиль шрифта для текста.

font-style: normal; /* Обычный стиль текста */
font-style: italic; /* Курсивный стиль текста */
font-style: oblique; /* Наклонный стиль текста */

font-family

Определяет шрифт для текста.

font-family: Arial, sans-serif; /* Устанавливает шрифт Arial
 с запасным вариантом sans-serif */
font-family: 'Times New Roman', serif; /* Устанавливает шрифт
 Times New Roman с запасным вариантом serif */

filter

Применяет графические эффекты, такие как размытие или изменение цвета к элементу.

filter: blur(5px); /* Применяет эффект размытия */
filter: brightness(0.5); /* Уменьшает яркость */
filter: contrast(200%); /* Увеличивает контрастность */

border-radius

Скругляет углы границы элемента.

border-radius: 10px; /* Устанавливает радиус скругления углов в 10 пикселей */
border-radius: 50%; /* Создает круг, если элемент квадратный */
border-radius: 10px 20px 30px 40px; /* Устанавливает разные радиусы для каждого угла */

width

Задает ширину элемента.

width: 100px; /* Устанавливает ширину в 100 пикселей */
width: 50%; /* Устанавливает ширину в 50% от родительского элемента */
width: auto; /* Автоматическая ширина, основанная на содержимом */

height

Задает высоту элемента.

height: 100px; /* Устанавливает высоту в 100 пикселей */
height: 50%; /* Устанавливает высоту в 50% от родительского элемента */
height: auto; /* Автоматическая высота, основанная на содержимом */

min-width

Задает минимальную ширину элемента.

min-width: 100px; /* Устанавливает минимальную ширину в 100 пикселей */
min-width: 50%; /* Устанавливает минимальную ширину в 50% от родительского элемента */

min-height

Задает минимальную высоту элемента.

min-height: 100px; /* Устанавливает минимальную высоту в 100 пикселей */
min-height: 50%; /* Устанавливает минимальную высоту в 50% от родительского элемента */

max-width

Задает максимальную ширину элемента.

max-width: 100px; /* Устанавливает максимальную ширину в 100 пикселей */
max-width: 50%; /* Устанавливает максимальную ширину в 50% от родительского элемента */

max-height

Задает максимальную высоту элемента.

max-height: 100px; /* Устанавливает максимальную высоту в 100 пикселей */
max-height: 50%; /* Устанавливает максимальную высоту в 50% от родительского элемента */

background-image

Устанавливает фоновое изображение для элемента.

background-image: url('image.jpg или ссылка'); /* Устанавливает фоновое изображение */

В поле ввода вы можете вставить ссылку на фото.

background-color

Устанавливает фоновый цвет для элемента.

background-color: blue; /* Устанавливает синий фоновый цвет */
background-color: #00FF00; /* Устанавливает фоновый цвет
 с использованием шестнадцатеричного значения */
background-color: rgba(0, 0, 255, 0.5); /* Устанавливает фоновый цвет
 с использованием значения RGBA с 50% непрозрачностью */

position

Позволяет задавать тип позиционирования элемента. Доступны следующие значения:

position: absolute; /* Абсолютное позиционирование от краев страницы */
.class {
  position: absolute;
  top: 10px;
}

position: relative; /* Относительное позиционирование
 от начального местоположения */
.class {
  position: relative;
  left: 10px;
}

position: fixed; /* Фиксированное позиционирование
 от краев страницы, не реагирует на прокрутку */
.class {
  position: fixed;
  bottom: 100px;
  z-index: 1; /* Порядок наложения */
}

position: sticky; /* Фиксированное позиционирование,
 останавливается в пределах родительского блока */
.class {
  position: sticky;
  top: 0;
}

Так как это сложное свойство, опробывать его вы можете на нашей песочнице.

margin

Устанавливает область отступа с четырех сторон элемента.

margin: 10px; /* Устанавливает отступ в 10 пикселей со всех сторон */
margin: 10px 20px; /* Устанавливает вертикальные и горизонтальные отступы */
margin: 10px 20px 30px 40px; /* Устанавливает отступы 
 для верха, справа, низа и слева */

padding

Устанавливает область внутреннего отступа с четырех сторон элемента.

padding: 10px; /* Устанавливает внутренний отступ
 в 10 пикселей со всех сторон */
padding: 10px 20px; /* Устанавливает вертикальные
 и горизонтальные внутренние отступы */
padding: 10px 20px 30px 40px; /* Устанавливает внутренние отступы
 для верха, справа, низа и слева */

Flexbox

Модель раскладки для упорядочивания элементов предсказуемым образом.

display: flex; Включает раскладку flexbox.

.container {
  display: flex;
}

flex-direction Определяет направление главной оси.

.container {
  flex-direction: row; /* Направление по умолчанию */
  flex-direction: row-reverse; /* Обратное направление */
  flex-direction: column; /* Вертикальная раскладка */
  flex-direction: column-reverse; /* Обратная вертикальная раскладка */
}

justify-content Выравнивает элементы вдоль главной оси.

.container {
  justify-content: flex-start; /* По умолчанию */
  justify-content: flex-end; /* К концу контейнера */
  justify-content: center; /* По центру */
  justify-content: space-between; /* С равными промежутками между элементами */
  justify-content: space-around; /* С равными промежутками вокруг элементов */
  justify-content: space-evenly; /* С равномерными промежутками */
}

align-items Выравнивает элементы вдоль поперечной оси.

.container {
  align-items: stretch; /* По умолчанию */
  align-items: flex-start; /* К началу контейнера */
  align-items: flex-end; /* К концу контейнера */
  align-items: center; /* По центру */
  align-items: baseline; /* По базовой линии текста */
}

align-self Выравнивает отдельный flex-элемент вдоль поперечной оси.

.item {
  align-self: auto; /* Наследует значение align-items */
  align-self: stretch; /* По умолчанию */
  align-self: flex-start; /* К началу контейнера */
  align-self: flex-end; /* К концу контейнера */
  align-self: center; /* По центру */
  align-self: baseline; /* По базовой линии текста */
}

flex-wrap Определяет, как элементы будут переноситься на новую строку.

.container {
  flex-wrap: nowrap; /* По умолчанию */
  flex-wrap: wrap; /* Перенос на новую строку */
  flex-wrap: wrap-reverse; /* Обратный перенос */
}

align-content Выравнивает строки элементов вдоль поперечной оси.

.container {
  align-content: stretch; /* По умолчанию */
  align-content: flex-start; /* К началу контейнера */
  align-content: flex-end; /* К концу контейнера */
  align-content: center; /* По центру */
  align-content: space-between; /* С равными промежутками между строками */
  align-content: space-around; /* С равными промежутками вокруг строк */
  align-content: space-evenly; /* С равномерными промежутками */
}

flex-grow Определяет, как элемент будет увеличиваться в размерах относительно других.

.item {
  flex-grow: 1; /* Растет пропорционально */
}

flex-shrink Определяет, как элемент будет уменьшаться в размерах относительно других.

.item {
  flex-shrink: 1; /* Уменьшается пропорционально */
}

flex-basis Определяет начальный размер элемента.

.item {
  flex-basis: 100px; /* Начальный размер элемента */
}

Так как это сложное свойство, опробывать его вы можете на нашей песочнице.

animation

Создание анимации

@keyframes имяАнимации{
  from{                        /* Начало */
    background-color: #1f7ecc;
  }
  50%{                               /* Середина */
    background-color: darkslateblue;
  }
  to{                          /* Конец */
    background-color: #1f7ecc;
  }
}

и её использование, а также задача параметров.

.class {
  animation-name: имяАнимации;
  animation-duration: 1s; /* Длительность */
  animation-timing-function: linear; /* Скорость(linear-постоянная,
   ease-замедление=>ускорение=>замедление,
    ease-in-медленно=>ускорение, ease-out-быстро=>замедление) */
  animation-delay: 1s; /* Задержка */
  animation-iteration-count: infinite; /* Кол-во повторение(infinite-бесконечно) */
  animation-direcrion: normal/reverse; /* Направление */
  animation-play-state: running/paused; /* Состояние анимации */

  animation: имяАнимации 1s lunear 0ms infinite reverse paused; /* Сокращенная формула */
}

transform

Применяет 2D или 3D трансформацию к элементу.

transform: translate(50px, 100px); /* Перемещает элемент */
transform: rotate(45deg); /* Поворачивает элемент */
transform: scale(2); /* Масштабирует элемент */
transform: skew(20deg, 10deg); /* Наклоняет элемент */

visibility

Определяет видимость элемента.

visibility: visible; /* По умолчанию, элемент виден */
visibility: hidden; /* Элемент скрыт, но занимает место в макете */

opacity

Устанавливает уровень прозрачности элемента.

opacity: 1; /* По умолчанию, полностью непрозрачно */
opacity: 0.5; /* 50% непрозрачность */
opacity: 0; /* Полностью прозрачно */

Псевдоэлементы

Используются для стилизации определенных частей элемента.

::before; /* Вставляет контент перед элементом */
.example::before {
  content: "Префикс";
}

::after; /* Вставляет контент после элемента */
.example::after {
  content: "Суффикс";
}

::first-line; /* Стилизует первую строку элемента */
.example::first-line {
  font-weight: bold;
}

::first-letter; /* Стилизует первую букву элемента */
.example::first-letter {
  font-size: 2em;
  color: red;
}

Так как это сложное свойство, опробывать его вы можете на нашей песочнице.

Псевдоклассы

Используются для определения особого состояния элемента.

:hover; /* Стилизует элемент при наведении курсора */
.example:hover {
  color: blue;
  animation-play-state: runnung; /* Запуск анимации */
}

:focus; /* Стилизует элемент, когда он получает фокус */
.example:focus {
  outline: 2px solid green;
}

:nth-child(n); /* Стилизует n-й дочерний элемент */
.example:nth-child(2) {
  background-color: yellow;
}

:not(selector); /* Стилизует элемент, не являющийся указанным селектором */
.example:not(.excluded) {
  color: gray;
}

Так как это сложное свойство, опробывать его вы можете на нашей песочнице.
